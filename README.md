# Chau Duc Manh - Portfolio v2

Đây là website portfolio cá nhân của mình, xây dựng để giới thiệu hồ sơ nghề nghiệp, kỹ năng, kinh nghiệm và các dự án tiêu biểu.

## About Me

- Fresher Software Engineer với gần 1 năm kinh nghiệm thực tế phát triển ứng dụng web.
- Mong muốn làm việc và cống hiến trong môi trường đa văn hóa, định hướng toàn cầu, đặc biệt ở các team hợp tác bằng tiếng Anh và tiếng Hàn.
- Mục tiêu dài hạn: phát triển theo hướng DevOps Engineer hoặc kỹ sư cầu nối Việt - Hàn.
- Sinh viên ngành Công nghệ thông tin tại Trường Đại học Công nghệ Thông tin - ĐHQG TP.HCM (UIT).
- TOEIC 725.

## Portfolio Includes

- Hero + Career Objective
- Education
- Technical Skills + Soft Skills
- Experience (gộp theo công ty, tách theo chức vụ)
- Projects (có links Website/GitHub)
- Certificates
- Hỗ trợ song ngữ Việt - Anh
- Responsive trên desktop/tablet/mobile
- Dark mode / Light mode

## Tech Stack

- React 18
- TypeScript
- Vite 6
- Tailwind CSS 4
- Lucide React

## Run Locally

```bash
npm install

```

## Update Your Profile Content

Toàn bộ dữ liệu hồ sơ được quản lý tập trung tại:

- `src/app/data/portfolio.ts`

Bạn có thể chỉnh trực tiếp các phần:

- `objective`
- `education`
- `technicalSkills`
- `experience`
- `projects`
- `certificates`
- `contact`
